# DevBox Runtime 文档

## 概述

当前仓库的正式结构由四部分组成：

- `tooling/`: `tooling` 镜像的构建上下文
- `base-images/`: 基础镜像定义
- `runtime-images/`: 最终运行时定义
- `tests/runtime-smoke/`: 运行时 smoke tests
