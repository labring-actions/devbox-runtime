# Go HTTP 服务器示例

这是一个简单的 Go Web 服务器示例，演示了基本的 HTTP 服务器功能。

## 项目描述

该项目使用 Go 标准库的 `net/http` 包创建了一个轻量级的 HTTP 服务器。服务器监听 `8080` 端口，访问时返回 “Hello, World!” 消息。项目支持开发和生产两种运行模式。

## 运行环境

该项目在预配置了 Go 的 Debian 12 系统上运行（Devbox 环境）。您无需手动设置 Go 或系统依赖。开发环境包含构建和运行 Go 应用所需的所有工具。如需针对特定需求进行调整，可修改相应的配置文件。

## 项目执行

**开发模式：** 在常规开发环境中，进入 Devbox 并在终端运行 `bash entrypoint.sh`。该命令会使用 `go run` 直接运行应用程序。

**生产模式：** 发布后，项目会根据 `entrypoint.sh` 脚本使用生产参数自动打包为 Docker 镜像并部署（运行 `bash entrypoint.sh production`）。该流程会构建可执行二进制并运行之。

DevBox: 编码、构建、部署，我们来打理其余工作。

使用 DevBox，您可以专注于编写优秀代码，其余基础设施、扩展和部署交由我们处理。从开发到生产的无缝体验。
