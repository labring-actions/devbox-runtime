# Angular Web 应用示例

这是一个使用 Angular v18 构建的现代 Web 应用示例，Angular 是一个专注于性能和开发效率的平台和框架，用于使用 HTML、TypeScript 和 CSS 构建客户端应用。

## 项目描述

该项目使用 Angular 创建一个单页应用。该应用展示了 Angular 的核心功能，包括其基于组件的架构、强大的依赖注入系统和全面的工具链。开发服务器监听 4200 端口，并提供热重载功能，提供流畅的开发体验。Angular 的强大功能使其非常适合构建具有可维护代码结构和优化性能的动态企业级应用。

## 运行环境

该项目在带有 Node.js 和 Angular v18 的 Debian 12 系统上运行，这些环境已在 Devbox 中预配置。您无需担心自己设置 Node.js、npm 或 Angular 依赖项。开发环境包含构建和运行 Angular 应用所需的所有工具。如果您需要进行调整以满足特定需求，可以相应地修改配置文件。

## 项目执行

**开发模式：** 对于正常开发环境，直接进入 Devbox 并在终端中运行 `bash entrypoint.sh`。这将启动带有热重载功能的 Angular 开发服务器。

**生产模式：** 发布后，项目将根据 `entrypoint.sh` 脚本和命令参数自动打包成 Docker 镜像并部署（运行 `bash entrypoint.sh production`）。这将安装依赖、构建优化的生产包，并使用 `serve` 静态文件服务器在 4200 端口提供服务。
