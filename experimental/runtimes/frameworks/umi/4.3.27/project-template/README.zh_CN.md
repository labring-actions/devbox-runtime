# UmiJS Web 应用程序示例

这是一个使用 UmiJS 4.3.27 构建的现代 React 应用程序示例，UmiJS 是一个强大的 React 框架，开箱即用地包含路由、状态管理和构建工具。

## 项目描述

该项目使用 UmiJS 框架创建一个单页应用程序。该应用包括主页和文档页面之间的基本路由，具有干净的开发体验。UmiJS 提供约定优于配置，因此项目结构以标准化方式组织。开发服务器提供热模块替换功能，提供流畅的开发体验。

## 运行环境

该项目在带有 Node.js 和 UmiJS 4.3.27 的 Debian 12 系统上运行，这些环境已在 Devbox 中预配置。您无需担心自己设置 Node.js、npm 或 UmiJS 依赖项。开发环境包含使用 UmiJS 构建和运行 React 应用程序所需的所有工具，包括用于快速开发和优化构建的 Vite。如果您需要进行调整以满足特定需求，可以相应地修改配置文件。

## 项目执行

**开发模式：** 对于正常开发环境，直接进入 Devbox 并在终端中运行 `bash entrypoint.sh`。这将安装依赖并启动带有热重载功能的 UmiJS 开发服务器。

**生产模式：** 发布后，项目将根据 `entrypoint.sh` 脚本和命令参数自动打包成 Docker 镜像并部署（运行 `bash entrypoint.sh production`）。这将安装依赖、构建优化的静态文件并使用 UmiJS 生产服务器提供服务。
