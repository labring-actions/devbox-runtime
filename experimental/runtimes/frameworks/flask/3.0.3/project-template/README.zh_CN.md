# Flask Web 应用程序示例

这是一个使用 Flask 3.0.3 构建的现代 Python Web 应用程序示例，Flask 是一个轻量级 WSGI Web 应用程序框架，旨在快速轻松地开始，并能够扩展到复杂的应用程序。

## 项目描述

该项目使用 Flask 创建一个轻量级 Web 服务。该应用展示了 Flask 的核心功能，包括其简单的路由系统和开发服务器。服务器监听 8080 端口，提供一个基本端点，返回"Hello World!"消息。Flask 的极简设计和灵活性使其非常适合小型项目和大型应用程序，拥有广泛的扩展生态系统，用于添加数据库集成、表单验证和身份验证等功能。

## 运行环境

该项目在带有 Python 和 Flask 3.0.3 的 Debian 12 系统上运行，这些环境已在 Devbox 中预配置。您无需担心自己设置 Python、虚拟环境或 Flask 依赖项。开发环境包含构建和运行 Flask 应用程序所需的所有工具。如果您需要进行调整以满足特定需求，可以相应地修改配置文件。

## 项目执行

**开发模式：** 对于正常开发环境，直接进入 Devbox 并在终端中运行 `bash entrypoint.sh`。这将创建虚拟环境、安装 Flask 依赖并启动 Flask 开发服务器，启用调试模式以自动重载。

**生产模式：** 发布后，项目将根据 `entrypoint.sh` 脚本和命令参数自动打包成 Docker 镜像并部署（运行 `bash entrypoint.sh production`）。这将创建虚拟环境、安装 Flask 和 Gunicorn 依赖，并使用 Gunicorn 作为生产 WSGI 服务器运行应用程序，使用工作进程以获得更好的性能和可靠性。
