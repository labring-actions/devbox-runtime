# Hugo 静态站点示例

这是一个使用 Hugo v0.135.0 构建的现代静态站点示例，Hugo 是一个用 Go 编写的快速灵活的静态站点生成器，以其速度、简单性和强大的内容管理功能而闻名。

## 项目描述

该项目使用 Hugo 创建一个静态网站。该应用展示了 Hugo 的核心功能，包括内容管理、模板系统和主题定制。Hugo 的极快构建过程允许您生成可以部署到任何 Web 服务器或 CDN 的静态 HTML 文件。开发服务器监听 1313 端口，并提供实时重载功能，提供流畅的内容创建体验。Hugo 的以内容为中心的方法使其非常适合博客、文档站点、作品集和企业网站。

## 运行环境

该项目在带有 Hugo v0.135.0 的 Debian 12 系统上运行，这些环境已在 Devbox 中预配置。您无需担心自己设置 Hugo、Go 或任何依赖项。开发环境包含构建和运行 Hugo 站点所需的所有工具。如果您需要进行调整以满足特定需求，可以相应地修改配置文件。

## 项目执行

**开发模式：** 对于正常开发环境，直接进入 Devbox 并在终端中运行 `bash entrypoint.sh`。这将启动带有实时重载功能的 Hugo 开发服务器。

**生产模式：** 发布后，项目将根据 `entrypoint.sh` 脚本和命令参数自动打包成 Docker 镜像并部署（运行 `bash entrypoint.sh production`）。这将构建优化的静态文件（启用压缩）并使用 Hugo 的内置服务器提供服务。
