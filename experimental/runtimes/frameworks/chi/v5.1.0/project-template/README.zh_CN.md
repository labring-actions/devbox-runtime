# Chi Web 应用程序示例

这是一个使用 Chi v5.1.0 构建的现代 Go Web 应用程序示例，Chi 是一个轻量级、符合 Go 习惯且可组合的路由器，用于构建具有清晰、优雅模式和中间件支持的 Go HTTP 服务。

## 项目描述

该项目使用 Chi 创建一个轻量级 Web 服务器。该应用展示了 Chi 的核心功能，包括其轻量级路由器、中间件系统和清晰的请求处理模式。服务器监听 8080 端口，提供一个简单的端点，返回"欢迎"消息。Chi 的极简设计，基于标准 net/http 库构建，无需外部依赖，使其非常适合使用干净、可维护的代码构建高性能 RESTful 服务和 Web 应用程序。

## 运行环境

该项目在带有 Go 和 Chi v5.1.0 的 Debian 12 系统上运行，这些工具已在 Devbox 环境中预配置。您无需担心自己设置 Go、依赖项或 Chi 框架。开发环境包含构建和运行 Go 应用程序所需的所有工具。如果您需要进行调整以满足特定需求，可以相应地修改配置文件。

## 项目执行

**开发模式：** 对于正常开发环境，直接进入 Devbox 并在终端中运行 `bash entrypoint.sh`。这将安装依赖并直接运行 Chi 应用程序源代码，使用 `go run main.go`。

**生产模式：** 发布后，项目将根据 `entrypoint.sh` 脚本和命令参数自动打包成 Docker 镜像并部署（运行 `bash entrypoint.sh production`）。这将安装依赖、构建优化的可执行二进制文件并运行应用程序。
