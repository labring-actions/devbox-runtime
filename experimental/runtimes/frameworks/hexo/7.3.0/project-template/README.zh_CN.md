# Hexo 博客示例

这是一个使用 Hexo v7.3.0 构建的现代静态站点生成器，Hexo 是一个快速、简单、高效的博客框架，由 Node.js 驱动，为构建强大的网站和博客提供强大的功能集。

## 项目描述

该项目使用 Hexo 创建一个静态网站/博客。该应用展示了 Hexo 的核心功能，包括内容管理、主题定制和高效的静态站点生成。开发服务器默认监听 4000 端口，并提供完整的博客结构，包括文章、页面和可定制主题。Hexo 的干净设计和以性能为中心的架构使其非常适合使用最少的配置构建快速加载的网站和博客。

## 运行环境

该项目在带有 Node.js 和 Hexo v7.3.0 的 Debian 12 系统上运行，这些环境已在 Devbox 中预配置。您无需担心自己设置 Node.js、依赖项或 Hexo 框架。开发环境包含构建和运行 Hexo 应用程序所需的所有工具。如果您需要进行调整以满足特定需求，可以相应地修改配置文件，如 `_config.yml`。

## 项目执行

**开发模式：** 对于正常开发环境，直接进入 Devbox 并在终端中运行 `bash entrypoint.sh`。这将安装依赖并启动 Hexo 开发服务器，允许您在 http://localhost:4000 预览您的站点。

**生产模式：** 发布后，项目将根据 `entrypoint.sh` 脚本和命令参数自动打包成 Docker 镜像并部署（运行 `bash entrypoint.sh production`）。这将安装依赖、生成静态文件并使用 http-server 提供服务。
