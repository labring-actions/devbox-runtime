# Nuxt3 Web 应用程序示例

这是一个使用 Nuxt3 v3.13 构建的现代 Vue.js 全栈应用程序示例，Nuxt3 是一个基于 Vue 3 的强大框架，提供服务器端渲染、静态站点生成和丰富的开发体验。

## 项目描述

该项目使用 Nuxt3 创建一个单页应用程序。该应用展示了 Nuxt3 的核心功能，包括自动路由生成、服务器端渲染、API 集成和静态站点生成功能。开发服务器监听 3000 端口，并提供热重载功能，提供流畅的开发体验。项目结构遵循 Nuxt3 的约定式设计，简化开发过程并提高生产力。

## 运行环境

该项目在带有 Node.js 和 Nuxt3 v3.13 的 Debian 12 系统上运行，这些环境已在 Devbox 中预配置。您无需担心自己设置 Node.js、pnpm 或 Nuxt 依赖项。开发环境包含构建和运行 Nuxt 应用程序所需的所有工具。如果您需要进行调整以满足特定需求，可以相应地修改配置文件。

## 项目执行

**开发模式：** 对于正常开发环境，直接进入 Devbox 并在终端中运行 `bash entrypoint.sh`。这将安装依赖并启动带有热重载功能的 Nuxt 开发服务器。

**生产模式：** 发布后，项目将根据 `entrypoint.sh` 脚本和命令参数自动打包成 Docker 镜像并部署（运行 `bash entrypoint.sh production`）。这将安装依赖、构建优化的静态文件并启动 Nuxt 生产服务器。
