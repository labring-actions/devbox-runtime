# Vue.js Web 应用程序示例

这是一个现代 Vue.js 3 前端应用程序示例，展示了使用 Vite 的基本 Web 应用程序设置。

## 项目描述

该项目使用 Vue.js 3 和 Vite 创建一个响应式单页应用程序。该应用展示了 Vue 的组件结构、响应式系统和样式功能。开发服务器监听 3000 端口，并提供热模块替换功能，提供流畅的开发体验。

## 运行环境

该项目在带有 Node.js 和 Vue.js 3.4.29 的 Debian 12 系统上运行，这些环境已在 Devbox 中预配置。您无需担心自己设置 Node.js、npm 或 Vue 依赖项。开发环境包含构建和运行 Vue 应用程序所需的所有工具，包括用于快速开发和优化构建的 Vite。如果您需要进行调整以满足特定需求，可以相应地修改配置文件。

## 项目执行

**开发模式：** 对于正常开发环境，直接进入 Devbox 并在终端中运行 `bash entrypoint.sh`。这将安装依赖并启动带有热重载功能的 Vite 开发服务器。

**生产模式：** 发布后，项目将根据 `entrypoint.sh` 脚本和命令参数自动打包成 Docker 镜像并部署（运行 `bash entrypoint.sh production`）。这将安装依赖、构建优化的静态文件并使用 Vite 的预览服务器提供服务。
