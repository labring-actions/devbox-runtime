# Next.js Web 应用程序示例

这是一个使用 Next.js 14.2.5 构建的现代 React 框架应用程序示例，Next.js 是一个强大的全栈框架，支持服务器端渲染、静态站点生成和带有 React 的 API 路由。

## 项目描述

该项目使用 Next.js 创建一个服务器渲染的 React 应用程序。该应用展示了 Next.js 的核心功能，包括其基于文件的路由系统、服务器组件、图像优化和 CSS 模块。开发服务器监听 3000 端口，并提供热重载功能，提供流畅的开发体验。Next.js 的混合渲染功能允许通过服务器端渲染和静态站点生成来优化性能，使其非常适合 SEO 友好的 Web 应用程序。

## 运行环境

该项目在带有 Node.js 和 Next.js 14.2.5 的 Debian 12 系统上运行，这些环境已在 Devbox 中预配置。您无需担心自己设置 Node.js、pnpm 或 Next.js 依赖项。开发环境包含构建和运行 Next.js 应用程序所需的所有工具。如果您需要进行调整以满足特定需求，可以相应地修改配置文件。

## 项目执行

**开发模式：** 对于正常开发环境，直接进入 Devbox 并在终端中运行 `bash entrypoint.sh`。这将安装依赖并启动带有热重载功能的 Next.js 开发服务器。

**生产模式：** 发布后，项目将根据 `entrypoint.sh` 脚本和命令参数自动打包成 Docker 镜像并部署（运行 `bash entrypoint.sh production`）。这将安装依赖、构建优化的生产构建并启动 Next.js 生产服务器。
