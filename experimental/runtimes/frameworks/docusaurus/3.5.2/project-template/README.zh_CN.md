# Docusaurus 文档站点示例

这是一个使用 Docusaurus 3.5.2 构建的现代文档站点示例，Docusaurus 是一个静态站点生成器，专为创建文档网站而设计，专注于易用性、可定制性和 SEO 友好的输出。

## 项目描述

该项目使用 Docusaurus 创建一个全面的文档站点。该应用展示了 Docusaurus 的核心功能，包括文档版本控制、博客功能、国际化支持和 MDX 集成。开发服务器监听 3000 端口，并提供热重载功能，提供流畅的内容编辑体验。Docusaurus 的强大功能使其非常适合创建文档站点、技术博客和产品落地页，具有干净、一致的导航和出色的搜索引擎优化。

## 运行环境

该项目在带有 Node.js 和 Docusaurus 3.5.2 的 Debian 12 系统上运行，这些环境已在 Devbox 中预配置。您无需担心自己设置 Node.js、npm 或 Docusaurus 依赖项。开发环境包含构建和运行 Docusaurus 站点所需的所有工具。如果您需要进行调整以满足特定需求，可以相应地修改配置文件。

## 项目执行

**开发模式：** 对于正常开发环境，直接进入 Devbox 并在终端中运行 `bash entrypoint.sh`。这将安装依赖并启动带有热重载功能的 Docusaurus 开发服务器。

**生产模式：** 发布后，项目将根据 `entrypoint.sh` 脚本和命令参数自动打包成 Docker 镜像并部署（运行 `bash entrypoint.sh production`）。这将安装依赖、构建优化的生产静态站点并提供服务。
