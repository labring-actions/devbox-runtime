# Sealaf Web 应用程序框架示例

这是一个使用 Sealaf 1.0.0 构建的现代基于函数的 Web 应用程序框架示例，提供干净而强大的 API 开发体验。

## 项目描述

该项目创建一个轻量级函数式编程 Web 服务框架，展示 Simple Web 的文件系统路由功能。该应用使用函数式方法编写后端 API，具有零配置和自动路由生成。服务器监听 2342 端口，提供干净、直观的 API 开发体验。该框架支持 WebSocket、XML 解析、CORS 和其他功能，使其适合快速开发小型应用程序、微服务和云函数。

## 运行环境

该项目在带有 Node.js 和 Sealaf 1.0.0 的 Debian 12 系统上运行，这些环境已在 Devbox 中预配置。您无需担心自己设置 Node.js、pnpm 或 Sealaf 依赖项。开发环境包含构建和运行 Sealaf 应用程序所需的所有工具。如果您需要进行调整以满足特定需求，可以相应地修改配置文件。

## 项目执行

**开发模式：** 对于正常开发环境，直接进入 Devbox 并在终端中运行 `bash entrypoint.sh`。这将安装依赖并启动带有热重载功能的 Simple Web 开发服务器。

**生产模式：** 发布后，项目将根据 `entrypoint.sh` 脚本和命令参数自动打包成 Docker 镜像并部署（运行 `bash entrypoint.sh production`）。这将安装依赖、构建优化的生产构建并运行应用程序。
