# Rocket Web 应用程序示例

这是一个使用 Rocket 0.5.1 构建的现代 Rust Web 应用程序示例，Rocket 是一个快速、安全的 Web 框架，使编写可靠的 Web 应用程序变得简单，而不会牺牲灵活性、可用性或类型安全。

## 项目描述

该项目使用 Rocket 创建一个轻量级 Web 服务器。该应用展示了 Rocket 的路由系统和配置功能。服务器监听所有网络接口（0.0.0.0），在根路径访问时响应"Hello, world!"。Rocket 的类型安全 Web 开发方法确保强大的错误处理和请求验证，而其简单的语法允许干净、可维护的代码。

## 运行环境

该项目在带有 Rust 和 Rocket 0.5.1 的 Debian 12 系统上运行，这些工具已在 Devbox 环境中预配置。您无需担心自己设置 Rust、Cargo 或 Rocket 依赖项。开发环境包含构建和运行 Rust 应用程序所需的所有工具。如果您需要进行调整以满足特定需求，可以相应地修改配置文件。

## 项目执行

**开发模式：** 对于正常开发环境，直接进入 Devbox 并在终端中运行 `bash entrypoint.sh`。这将使用 Cargo 的开发模式构建并运行 Rocket 应用程序。

**生产模式：** 发布后，项目将根据 `entrypoint.sh` 脚本和命令参数自动打包成 Docker 镜像并部署（运行 `bash entrypoint.sh production`）。这将构建优化的发布二进制文件并运行应用程序。
