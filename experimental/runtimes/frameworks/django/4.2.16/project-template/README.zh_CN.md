# Django Web 应用程序示例

这是一个使用 Django 4.2.16 构建的现代 Python Web 应用程序示例，Django 是一个高级 Web 框架，通过其"内置电池"理念鼓励快速开发和干净、实用的设计。

## 项目描述

该项目使用 Django 创建一个全面的 Web 应用程序。该应用遵循 Django 的模型-视图-模板（MVT）架构，展示了框架的核心功能，包括其强大的 ORM、自动管理界面和强大的安全系统。服务器监听 8000 端口，并使用 SQLite 作为默认数据库。Django 的广泛内置功能使其非常适合使用更少的代码和更多的灵活性构建复杂的、数据库驱动的网站和应用程序。

## 运行环境

该项目在带有 Python 和 Django 4.2.16 的 Debian 12 系统上运行，这些环境已在 Devbox 中预配置。您无需担心自己设置 Python、虚拟环境或 Django 依赖项。开发环境包含构建和运行 Django 应用程序所需的所有工具。如果您需要进行调整以满足特定需求，可以相应地修改配置文件。

## 项目执行

**开发模式：** 对于正常开发环境，直接进入 Devbox 并在终端中运行 `bash entrypoint.sh`。这将创建虚拟环境、安装 Django 依赖并启动 Django 开发服务器，使用增加的详细程度以获得更好的调试信息。

**生产模式：** 发布后，项目将根据 `entrypoint.sh` 脚本和命令参数自动打包成 Docker 镜像并部署（运行 `bash entrypoint.sh production`）。这将创建虚拟环境、安装 Django 依赖并在更类似生产的环境中启动 Django 应用程序。
