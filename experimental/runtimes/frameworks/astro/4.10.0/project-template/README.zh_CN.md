# Astro Web 应用程序示例

这是一个使用 Astro 4.10.0 构建的现代 Web 应用程序示例，Astro 是一个现代前端框架，通过提供最少的 JavaScript 和利用基于组件的架构来提供闪电般的性能。

## 项目描述

该项目使用 Astro 创建一个静态网站。该应用展示了 Astro 的核心功能，包括其基于组件的结构、基于文件的路由和部分水合功能。开发服务器监听 4321 端口，并提供热重载功能，提供流畅的开发体验。Astro 的"岛屿架构"允许选择性水合交互式组件，通过将大部分站点保持为静态 HTML 并仅向客户端提供最少的 JavaScript，从而实现优化的性能。

## 运行环境

该项目在带有 Node.js 和 Astro 4.10.0 的 Debian 12 系统上运行，这些环境已在 Devbox 中预配置。您无需担心自己设置 Node.js、npm 或 Astro 依赖项。开发环境包含构建和运行 Astro 应用程序所需的所有工具。如果您需要进行调整以满足特定需求，可以相应地修改配置文件。

## 项目执行

**开发模式：** 对于正常开发环境，直接进入 Devbox 并在终端中运行 `bash entrypoint.sh`。这将安装依赖并启动带有热重载功能的 Astro 开发服务器。

**生产模式：** 发布后，项目将根据 `entrypoint.sh` 脚本和命令参数自动打包成 Docker 镜像并部署（运行 `bash entrypoint.sh production`）。这将安装依赖、构建优化的生产静态站点，并使用预览服务器提供服务。
