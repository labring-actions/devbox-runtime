# Nginx Web 服务器示例

这是一个使用 Nginx 1.22.1 构建的现代 Web 服务器示例，Nginx 是一个高性能 HTTP 和反向代理服务器，以其稳定性、丰富的功能集和低资源消耗而闻名。

## 项目描述

该项目使用 Nginx 创建一个简单的 Web 服务器。该应用展示了 Nginx 的核心功能，包括静态文件服务、反向代理功能和高效的请求处理。服务器监听 80 端口并提供静态 HTML 文件。Nginx 的事件驱动架构使其非常适合为高流量网站提供服务以及作为应用服务器的反向代理。

## 运行环境

该项目在带有 Nginx 1.22.1 的 Debian 12 系统上运行，这些环境已在 Devbox 中预配置。您无需担心自己设置 Nginx 或配置服务器。开发环境包含运行 Nginx 所需的所有工具。如果您需要进行调整以满足特定需求，可以相应地修改配置文件。

## 项目执行

**开发模式：** 对于正常开发环境，直接进入 Devbox 并在终端中运行 `bash entrypoint.sh`。这将检查 Nginx 配置并以前台模式启动 Nginx 服务器。

**生产模式：** 发布后，项目将根据 `entrypoint.sh` 脚本和命令参数自动打包成 Docker 镜像并部署（运行 `bash entrypoint.sh production`）。这将检查 Nginx 配置并以前台模式启动 Nginx 服务器。
